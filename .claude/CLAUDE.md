# Global Rules

## Git & GitHub
- NEVER create or push fork repositories. All repos should be private.
- NEVER create public repositories or public gists without explicit permission.
- Always create gists as **secret** (`gh gist create` without `--public`).
- Always ask before any `git push`, `gh repo fork`, `gh repo create`, or similar commands.
- No changes, code, or content should ever be pushed to anything public.
- Always use **Conventional Commits** format: `type(scope): description` (e.g. `feat:`, `fix:`, `chore:`, `refactor:`, `docs:`, `style:`, `perf:`, `ci:`, `build:`, `test:`).
